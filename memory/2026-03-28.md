## Bug Pine Script — ternaire multi-ligne (15h13 AST)

### Erreur : "Mismatched input '?' expecting 'end of line without line continuation'"
- **Cause** : les ternaires ajoutés pour le compound avaient `?` en début de nouvelle ligne
- Pine Script v5 ne tolère pas `?` en début de ligne dans un ternaire
- **Règle permanente** : TOUJOURS écrire les ternaires sur UNE SEULE ligne en Pine Script v5
- **Lignes concernées** : `rawCapital` (L95) et `rawCapLab` (L145) dans ETH + SOL

### Fix appliqué (ETH)
```pine
// AVANT (cassé) :
rawCapital = useCompound
    ? math.max(...) : startCapital

// APRÈS (correct) :
rawCapital = useCompound ? math.max(...) : startCapital
```
- Idem pour `rawCapLab`
- Script ETH re-copié dans presse-papier ✅
- Script SOL : même fix à appliquer avant envoi

### État
- JP en train de coller la version corrigée dans TradingView
- En attente de confirmation que ça compile
